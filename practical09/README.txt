In this folder, there are several files for the ninth week practical.

Magic_square.txt shows a magic matrix. 

NotMagic_square.txt shows a non-magic matrix.

Magicsquare.h is the header, which is needed to run the programs.

Magicsquare.c reads the matrixs from the files and checks if they are magic.

Magicsquare_u.c reads the matrix which is input by user and checks if it is magic.
